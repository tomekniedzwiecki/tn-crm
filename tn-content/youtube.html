Konfiguracja YouTube API

Aby pobierać dane z YouTube, musisz skonfigurować klucz API.

  1. 1. Wejdź na Google Cloud Console
  2. 2. Utwórz projekt i włącz YouTube Data API v3
  3. 3. Utwórz klucz API (Credentials → Create Credentials → API Key)
  4. 4. Dodaj klucz do Supabase Secrets jako YOUTUBE_API_KEY
