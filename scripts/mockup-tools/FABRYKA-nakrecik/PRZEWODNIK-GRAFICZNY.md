## REGUŁA RYTMU (anti-szew)
- 2 sąsiednie sekcje NIE dzielą kontekst+skala: hero(kuchnia,wide) → problem(kuchnia,medium —
  ta sama scena ale INNA skala/emocja, dozwolone jako para ból↔ulga) → rozwiazanie(kuchnia,close)
  → demo(neutral set, close/stany — ZMIANA świata) → zastosowania(mix — max różnorodność) →
  tryby(neutral) → mid-cta(ulica wieczór) → final(park golden hour). Konteksty się rozjeżdżają.
- **⛔ ANTY-SZEW:** dwie sąsiednie sceny PEŁNOKADROWE nie mają obrazu po tej samej stronie —
  ZIG-ZAG: problem obraz PRAWA · rozwiazanie obraz LEWA · (demo/zastosowania = własne layouty) ·
  mid-cta i final = pełny dark-fallback rozdzielone sekcjami kodowymi (opinie/galeria/faq) → nie
  stykają się.
